> 深度优先搜索是，每次走一条路径到叶节点，然后回溯，再走下一条路径；
>
> 广度优先搜索则是一层一层对节点进行访问；
>
> 典型案例可以参考树的 前序遍历（dfs），层序遍历（bfs);
>

    def bfs(arr,start,target):
        used = [False]*len(arr)
        used[start] = True
        
        queue = [arr]
        while queue:
            cur = queue.pop()
            if cur==target: # 已经到达目标
                return True
            for i in range(len(arr)):
                if used[i]==False:
                    if Avaiable(arr[i]): # 判断 元素arr[i]是否为合法的下一个元素
                        queue.append(arr[i])
        
        return False
