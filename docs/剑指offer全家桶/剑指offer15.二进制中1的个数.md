剑指offer15.二进制中1的个数
----------
 - 题目
>请实现一个函数，输入一个整数（以二进制串形式），输出该数二进制表示中 1 的个数。例如，把 9 表示成二进制是 1001，有 2 位是 1。因此，如果输入 9，则该函数输出 2。
 - 示例
 ----------
>input: 00000000000000000000000000001011

> output: 3 
 ----------
 - 代码
 >
>
    class Solution:
        def hammingWeight(self, n: int) -> int:
            return collections.Counter(bin(n))['1']
